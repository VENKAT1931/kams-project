# Key Account Management System (KAMS)

## Overview
The Key Account Management System is a full-stack web application designed to manage high-value clients, track contracts, and monitor revenue.

## Live Links
- Frontend: https://kams-project.vercel.app
- Backend: https://kams-backend-b6cx.onrender.com

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot (Java)
- Database: H2
- Deployment: Render (Backend), Vercel (Frontend)

## Features
- Account Management
- Contract Management
- Dashboard Analytics (Revenue, Active, Expired)

## API Endpoints
- POST /accounts
- GET /accounts
- POST /contracts
- GET /contracts
- GET /dashboard

## How to Run Locally

### Backend
1. Open in IntelliJ
2. Run `KamsApplication.java`
3. Runs on http://localhost:8080
## frontend
1. Open `index.html`
2. Use Live Server or browser
